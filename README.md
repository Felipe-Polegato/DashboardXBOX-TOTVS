# Dashboard Excel – Análise de Dados (XBOX)

Este projeto foi desenvolvido como parte do desafio do curso **Fundamentos de Engenharia de Dados e Machine Learning**, oferecido pela TOTVS em parceria com a DIO (Digital Innovation One).

O objetivo foi aplicar conceitos de análise de dados e visualização utilizando o Excel.

---

## Objetivo

Criar um dashboard interativo para análise de dados relacionados ao universo XBOX, permitindo:

- Visualização clara de métricas
- Organização eficiente dos dados
- Geração de insights
- Prática de conceitos iniciais de dados

---

## Tecnologias Utilizadas

- Microsoft Excel
  - Tabelas dinâmicas
  - Gráficos
  - Segmentação de dados (Slicers)
  - Limpeza e organização de dados

---

## Estrutura do Projeto

O arquivo contém:

- Base de dados tratada  
- Tabelas dinâmicas  
- Visualizações gráficas  
- Dashboard interativo  

---

## Funcionalidades

- Filtros dinâmicos
- Gráficos interativos
- Indicadores visuais
